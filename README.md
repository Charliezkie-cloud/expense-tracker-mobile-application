## Tech Stack
- Framework: React Native (Expo)
- Programming Language: Typescript/Javascript
- Database Engine: Expo SQLite
- Target OS: Android

## Development
1. Download or clone the source code via git ``git clone https://github.com/Charliezkie-cloud/expense-tracker-mobile-application.git``
2. Install the packages ``npm install``
3. And run the expo development server ``npx expo start``
4. Enjoy ;D
